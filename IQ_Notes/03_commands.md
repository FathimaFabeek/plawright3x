# VS Code Commands

This file contains some commonly used Visual Studio Code commands in a table format for quick reference.

| Command | Shortcut | Purpose |
|--------|----------|---------|
| Quick Open | Ctrl + P | Open files quickly by name |
| Command Palette | Ctrl + Shift + P | Open commands, settings, and extensions |
| New File | Ctrl + N | Create a new file |
| Save File | Ctrl + S | Save the current file |
| Save All | Ctrl + Alt + S | Save all open files |
| Find | Ctrl + F | Search text inside the current file |
| Replace | Ctrl + H | Replace text in the current file |
| Search in Files | Ctrl + Shift + F | Search text across the whole workspace |
| Toggle Sidebar | Ctrl + B | Show or hide the Explorer sidebar |
| Open Terminal | Ctrl + ` | Open the integrated terminal |
| Split Editor | Ctrl + \ | Open the current file in a split view |
| Close Editor | Ctrl + W | Close the active tab |
| Comment/Uncomment | Ctrl + / | Toggle line comments |
| Format Document | Shift + Alt + F | Format the code automatically |
| Go to Line | Ctrl + G | Jump to a specific line number |
| Open Settings | Ctrl + , | Open VS Code settings |
| New Window | Ctrl + Shift + N | Open a new VS Code window |
| Zen Mode | Ctrl + K Z | Focus only on the editor |
